# Всё, что вы когда-либо хотели знать про Радио-86РК и его клоны

Список "живой" и постоянно обновляется. Принимаются пул-реквесты.

## Эмуляторы

### rk86.ru

[rk86.ru](https://rk86.ru) - Онлайновый эмулятор Александра Дёмина. Поддерживает только Радио-86РК, включая цвет.

Интегрированы:

- отладчик
- ассемблер (asm8)
- компилятор C (порт c8080)
- компилятор PL/M-80
- визуализатор магнитофонных записей РК
- снапшоты

Исходники: <https://github.com/begoon/rk86-js>

### 86rk.ru

[86rk.ru](https://86rk.ru) - Онлайновый эмулятор Виталия Поединка. Поддерживает несколько клонов Радио-86РК, включая цвет.

Интегрированы:

- отладчик
- дизассемблер
- редактор знакогенератора

Исходники: <https://github.com/radio-86rk/86rk.ru>

## emu80

[emu80.org](https://online.emu80.org) - Открытый мультиплатформенный эмулятор Виктора Пыхонина для старых советских бытовых компьютеров: «Радио-86РК», «Апогей», «Партнёр», «Микроша», «Микро-80», «ЮТ-88», «Электроника КР-04», «Орион-128», «Специалист», «Эврика», «Лик», «Специалист-MX», «Сура», «Веста», «Хобби», «Вектор-06Ц», «Львов», «Корвет».

Исходники: <https://github.com/vpyk/emu80v4>
